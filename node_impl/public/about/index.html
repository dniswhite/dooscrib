DooScrib
  • About

An idea that came from watching my wife play Draw Something and I was wondering if I could create a similar drawing pad on the net.

As time progresses and I work on this more then I plan to make it all open source.

Goals

  • Concept (Done)

    • Basic proof of concept with rough code.

  • jQuery plugin (Done - read article here.)

    • Create a jQuery plugin to emcompass the basic development already done as a proof of concept.

  • Shared Canvas (Currently in Development)

    • Redploy and integrate with a Node.js and Express solution using socket.io to enable a single shared canvas.

  • Groups and Rooms

    • Add abilities to create groups/rooms with a shared canvas.

  • More Features

    • Continue to add more features...

For now enjoy what you can get from it. You can get the plugin source here.

This site was designed by @dniswhite and for simplicity by using bootstrap.
